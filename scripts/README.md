| Script | Purpose |
|---|---|
| `run.sh` | Launch orchestrator + all workers |
| `run_orchestrator.py` | Start orchestrator only |

| `run_image_worker.py` | Start image worker only |
| `seed_demo.py` | Submit demo requests to test the pipeline |
